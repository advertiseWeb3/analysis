# 🍍社区网络机制

社区网络机制旨在通过激励和推荐的方式，建立一个强大的社区网络，为广告业务的发展奠定坚实基础。 机制鼓励用户积极参与社区，用努力和贡献获取合理的回报。同时，由已经形成一定社群规模的成员形成区块链节点，建设生态并分享红利，为Ad-Web3去中心化广告网络的运行与治理持续注入活力。

# 社区机制说明

# 1. 新会员加入

  • 每位新会员都将通过推荐人的邀请链接加入，最初会员级别为0级。

  • 新会员将被自动排在推荐人拓展的网络链路中，层数最短且注册时间最早的位置。

  • 推荐人网络每层点位数量为3ⁿ。

# 2. 会员晋级

  • 新会员升级一级时，需向推荐人的钱包地址转100USDT，作为推荐人奖励，则可晋级为1级会员，并获得自己的专属邀请链接。

  • 晋升n级会员，必须完成两个条件：

    • a. 成功邀请3个会员并晋级1级；
    • b. 晋升n级则给自己的上n层会员转100USDT。

  • 最高级为9级，并且仅在9层以内（包含9层）获得下级会员的转款。

# 3. 级别跳跃

  • 会员向上打款时，如相应层数的上级未达到相应级别，则会跳过向上查找第一个符合级别条件的会员，以此保障网络连续性。

  • 例如，某会员需要打款升级至5级，但其直接上5层会员仅为3级，则会跳过此会员，向上查找第一个5级会员。

# 4. 成为区块链网络节点

  • 成功达到9级的成员将成为Ad-Web3广告区块链节点。这些节点将在Ad-Web3代币系统与广告分发中发挥重要作用，在获得个人更多收益的同时，也为Ad-Web3广告业务的发展和社区的繁荣做出贡献。

  • 👉激活时切记用钱包bsc地址绑定不要用交易所地址，转账≥100，不要99.9！！

  • TIPS: 当你的1代用户已经满3且激活，又有很多朋友可以推荐时如何做？ 可以向自己的3个1代用户进行分配，帮助他们发展网络，由1代用户提供他们的邀请链接，发给你的第4个用户； 如果1代3个用户也直推满3且激活，可以帮助他们的1代也就是你的2代发展网络，由你2代用户提供邀请链接发给你的第4个用户。 以此类推便可以快速的形成帮扶裂变效应达到财富目标！

通过以上社区机制，Ad-Web3将建立一个活跃、稳健的社区网络，在web3时代，开辟广告业发展的新路径。
