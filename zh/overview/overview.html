# 🍅Ad-Web3即将爆发概述:

Ad-Web3是一个基于区块链的广告平台，利用智能合约实现广告位的去中心化交易。广告商可以使用平台代币购买广告空间，会员可以通过参与社区游戏获得代币奖励。平台不仅提供广告曝光，还通过互动和推广活动激励用户参与平台生态。

# 获得相关文档：

代币代码：ADW

总供应量：210亿

发行日期：2024.8

分配机制：基于社区的发行规则游戏：

  • 发送100u，用户获得10,000 ADW

  • 接收100u，用户获得5,000 ADW

  • 释放总量高达21 000 000 000

# 最终销毁：

截至UTC时间2024年11月24日0:00，如果没有释放任何剩余的代币，所有代币将被销毁。 广告位销售机制 广告位的销售通过智能合约进行管理。广告商使用 ADWEB 进行购买，合约自动执行以下操作：

  • 销毁：合约收到的 ADW 的 10% 将被销毁。

  • 广告互动奖励：5%将分配给广告互动奖励（媒体号关注、转推、互动），吸引更多商家上架广告。

  • 质押奖励：40% 将分配给质押 ADW的用户。

  • 给予者奖励：30%将分发给给予100Usdt的用户作为推广奖励。

  • 受赠者奖励：15%将作为推广奖励分发给给予100Usdt的用户。社区激励计划Ad-Web3独特的在线游戏机制旨在快速扩大社区。

# 规则如下：

全球市场统一为3的倍数，形成社交网络，如图所示。每个新会员都会通过推荐人的邀请链接加入，初始会员等级为0级。

  • 当新会员晋升为等级，需要给推荐人100 USDT。然后即可晋升为1级会员，并获得专属邀请链接。

  • 晋升1级及以上，需完成相应的推荐和捐赠任务，最高可达9级。达到等级后，即可领取属于自己等级的100 USDT。未达到等级的会员，将无法获得超过该等级对应等级的礼物，导致收益损失。
