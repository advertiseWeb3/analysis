# AD-WEB3

# 概述：

AD-Web3是一个基于区块链的 DAO（社区自治组织）的广告平台，旨在利用区块链智能合约实现广告位的去中心化交易。广告商可使用平台代币购买广告位，社区用户通过贡献行为价值直接获取代币收益。社区V9用户将成为该平台的超级Node成员，决策平台生态发展细则。当数千万用户自由进入这个社会性试验平台，又同时享受去中心化广告带来的丰厚收益以及实现点对点分布式广告网络的自发式裂变，成员需通过参与社群互赠激活并获得代币奖励，平台不仅提供广告曝光，还激励用户通过互动和推广活动参与到平台的生态中。

# 代币计划

  • Token代码: ADW (Web3广告币)
  • 发行总量: 21,000,000.000
  • 发布时间: 2024.8
  • 分配机制:
    • 基于社区互赠游戏的释放规则:
      • 每赠出100 USDT，用户获得10,000 ADW.
      • 每受赠100 USDT，用户获得5,000 ADW.
      • 直至21,000,000,000全部释放。
    • 末期销毁:
      • 截止2024年11月24日UTC 0时，如有剩余Token未释放，则全部销毁。

# 广告位出售机制：

广告位的出售通过智能合约来管理。 广告商需使用ADW进行购买，合约将自动执行以下操作:

  • 销毁: 合约收到的ADW中，10%将被销毁。
  • 广告互动奖励: 5%将分配给广告互动奖励（媒体号关注、转推、互动），吸引更多商家上架广告。
  • 质押奖励: 40%将分配给质押ADW的用户。
  • 赠予者奖励: 30%将作为推广奖励，分配给赠出100 USDT的用户。
  • 受赠者奖励: 15%将作为推广奖励，分配给受赠100 USDT的用户。

# 社区激励计划：

AD-Web3独特的互赠网游戏机制，以快速扩大社群规模，规则如下:

  • 全球市场统一按照一个3的倍数形成社群网络，见图左。
  • 每位新会员都将通过推荐人的邀请链接加入，最初会员级别为0级。
  • 新会员升级一级时，需向推荐人赠予100 USDT，则可晋级为1级会员，并获得自己的专属邀请链接。
  • 晋升1级以上级别，需完成相应推荐与赠予任务，层级不限。
  • 达到几级即可收取自己几层的100 USDT赠予。
  • 不达到级别的会员不获得旗下超过级别对应层数的赠予，产生错失收益。
  • 从第四层开始，允许误差比率晋级（图右）。

# 项目核心亮点

  • 代币100%投放社区，无募资，高通缩、全流通。
  • 推广无风险，无泡沫，可持续。
  • 用户最大投资100 USDT，赚U赚币赚广告费，三重收益，潜力巨大。
  • 全球一张网，多语言市场，网络拓展无边界。
  • 代币价值伴随社区扩大，潜力无限。
